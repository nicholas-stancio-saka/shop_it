## API Usage
'https://fakestoreapi.com/users' -> Create Account

'https://fakestoreapi.com/auth/login' -> Login

'https://fakestoreapi.com/users/$id' -> Get User (Combine with login)

'https://fakestoreapi.com/carts/1' -> Get Cart

'https://fakestoreapi.com/carts/1' -> Update Cart


'https://fakestoreapi.com/carts/1' -> Delete Cart

'https://fakestoreapi.com/products/categories' -> Get All Categories

'https://fakestoreapi.com/products/category/$categoryId' -> Get Products in Specific Category

'https://fakestoreapi.com/products' -> Get All Products

'https://fakestoreapi.com/products/$productId' -> Get Specific Product

'???' -> Add product to cart
